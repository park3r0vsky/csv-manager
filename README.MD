## Running Project

### 游 Clone repository

<br>

### 丘뙖잺 Backend

a. Go to backend directory

```
cd backend
```

b. Create venv

```
python -m venv venv_name
```

c. Activate venv

```
venv_name/Scripts/activate
```

d. Install dependecies

```
pip install -r requirements.txt
```

e. Make and run migrations:

```
python manage.py makemigrations
```

```
python manage.py migrate
```

f. Start server on port 8000:

```
python manage.py runserver
```

<br>

### 游꿛 Frontend

a. Go to frontend directory

```
cd frontend
```

b. Install dependencies

```
npm i
```

c. Start client on port 3000

```
npm start
```

<br>

## Endpoints

### 游릭 GET api/file/list

游닇 Gets list of uploaded files

<br>

### 游리 POST api/file

游닇 Uploads file

Example Body: `{file: [data.csv]}`

<br>

### 游댮 DELETE api/file

游닇 Deletes file from database

Example Body: `{id: 1}`

<br>

### 游릭 GET api/file/preview/[ID]

游닇 Previews file from given id - replace [ID] with number e.g. 1

<br>

### 游리 POST api/file/enrich/[ID]

游닇 Enriches file from given id - replace [ID] with number e.g. 1

<br>

### 游릭 GET api/file/columns/[ID]

游닇 Shows the columns from file from given id - replace [ID] with number e.g. 1

<br>
